# Spotify Trend Analysis

## Business Context / Problem Statement  
Spotify is one of the world’s leading music streaming platforms, where understanding user listening behavior and track popularity is critical for artists, record labels, and platform managers. Data-driven insights into streaming trends can help stakeholders optimize marketing strategies, improve playlist curation, and identify emerging genres or artists. This analysis aims to uncover key factors influencing track popularity and streaming growth, enabling better decision-making and business impact.

## Project Objectives  
- Identify top-performing tracks, artists, and genres based on streaming metrics  
- Analyze the impact of listening trends on overall platform engagement  
- Provide actionable recommendations for marketing teams, playlist curators, and artist managers  

## Data Description  
- **Source:** Spotify track and artist statistics (2020–2022)  
- **Period covered:** 2010-2023 streaming data  
- **Key fields:** Track name, artist, genre, release year, streams, likes, playlist appearances, duration, popularity score  
- **Size of dataset:** 1,000+ tracks across multiple genres  

## Analytical Approach  
- Data cleaning and preprocessing using Power BI  
- Exploratory data analysis (EDA) using Powe BI
- Data visualization and interactive dashboards using Power BI  

## Key Insights & Findings  
- Top-performing tracks are those with high playlist appearances and strong engagement metrics (likes, shares)  
- Emerging genres show rapid growth in streaming numbers, especially among younger demographics  
- Artist consistency (regular releases) is strongly correlated with sustained popularity  

## Impact on Business Decision-Making  
- Marketing teams can optimize promotional campaigns based on track and genre trends  
- Playlist curators can design targeted playlists to maximize user engagement  
- Record labels can make informed decisions about artist contracts and resource allocation  
- **Influenced KPIs:** track popularity score, monthly active users, streaming growth, revenue  

## Recommendations  
- Focus on promoting tracks with high playlist potential and strong engagement metrics  
- Invest in emerging genres to capture new audience segments  
- Monitor artist release frequency and consistency to sustain platform engagement
